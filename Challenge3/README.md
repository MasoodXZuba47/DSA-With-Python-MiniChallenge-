# Two Sum

This is a Python solution for the standard **Two Sum** problem. 

Given an array of integers and a target value, it finds the two numbers that add up to the target and returns their indices.

## How to Run
Run the Python script from your terminal:
```bash
python chellenge3.py
```

## Output
When you run the script, it will display the output in the terminal and also save the results into a file named **`a.so`** in the same folder.
